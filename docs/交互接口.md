## 全局
* front-end -> back-end
```vue
url:'site/status/'
//无传入数据，获得当前登录账户，保证页面刷新时系统正确性
```

* front-end -> back-end
```vue
res: {
	result:boolean,//是否登陆状态
	info:String,//若false，有错误信息
	id:String,//学号or教工号
	userType:String,//'学生'or'教师'or'管理员'
	userName:String
}
```

## 登录与注册

### 注册

==此处添加真实姓名==

* front-end -> back-end

```vue
url: '/site/register'
data: {
  'studentId':String,
  'userName':String,
  'password':String,
	'realName':String
}
```

- back-end -> front-end

```vue
res: {
	'result':boolean
	//true代表注册成功，false代表注册失败
	'info':String
	//解释说明信息
}
```

### 课表

#### 学生课表

* front-end -> back-end
```vue
mounted

url: 'Student_Schedule'
data: qs.stringify({
}), 
```

* back-end -> front-end

```vue
res: {
	res.data.schedule;
}

//schedule为一个二维数组，包含16个一维数组，为16周的课程元素,其中每一个数组包含6个元素，从1到6的time分别为1，2节 ... 11,12节，
每个元素的格式如下：
		{
            time: '1,2节',
            Monday: '编译技术 锌小子 (一)215',
            Tuesday: '',
            Wednesday: '',
            Thursday: '',
            Friday: '数据库系统原理 铑小子 新主楼M-101',
            Saturday: '',
            Sunday: ''
          },
通俗来说，每行包含特定时间段（第几节）周一到周日的全部课程，但需要后端限制time的值按顺序排列

```
